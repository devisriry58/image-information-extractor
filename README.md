# Image Information Extractor

This is a simple GitHub-hosted web app that allows you to:

1. Upload an image  
2. Extract text/information from the image using an AI API  
3. Download the extracted data as an Excel-compatible CSV file  

## How to Use

1. Upload all files to a new GitHub repository.
2. Go to **Settings â†’ Pages**.
3. Select Branch: `main` and folder: `/root`.
4. Your site will be live at:

```
https://yourusername.github.io/repository-name/
```

## Setup

Edit `script.js` and replace:

```
YOUR_OPENAI_API_KEY
```

with your own key.

That's it!
